Instructions

  • Zoom: Click the canvas (grid) Hold Ctrl + scroll mouse wheel
  • Place Blocks: Click a tile then click grid
  • Erase Blocks: Select the seed (eraser) then select tile or right-click on grid to erase
  • Save: You can save as PNG or world project (JSON)
  • Load: Load saved world files here

Collaboration

You can export your world as a JSON project file to share your builds. Others can load the JSON to build with you!
